# 版本規範

### 命名規則

OMFLOW版號分為4碼：

1. 第一碼&#x70BA;_&#x4E16;代更新_：當產品已更換成程式語言或以全新姿態推出時，會更新此碼。
2. 第二碼&#x70BA;_&#x91CD;大更新_：加入全新服務時，會更新此碼。
3. 第三碼&#x70BA;_&#x529F;能更新_：在既有的產品框架下新增或優化功能時會更新此碼。
4. 第四碼&#x70BA;_&#x932F;誤修正_：修正BUG會更新此碼。

### 更新規則

每次更新時，請先確認目前運行OMFLOW的版本號碼，更新幅度最多只&#x80FD;_&#x76F8;差一個功能更新版號(第三碼)_，如 1.2.1.X → 1.2.2.X。

在1.2.0之前的版本，請參考以下規則表：

| 更新檔版號   | 可更新的OMFLOW版號範圍    |
| ------- | ----------------- |
| 1.2.0.1 | 1.2.0.0 - 1.2.0.1 |
| 1.2.0.0 | 1.1.6.3           |
| 1.1.6.3 | 1.1.6.0 - 1.1.6.3 |

> 版號右方字符不影響更新規則，如1.1.6.3fix3

### Patch下載

須憑授權碼至 [產品支援網站](https://support.syscom.com.tw/) 進行註冊，註冊完成後即可下載

操作步驟如下：\
1\. 註冊，凌群電腦員工可使用公司信箱( syscom.com.tw )直接註冊，\
2\. 註冊授權

* 登入後於【左側主選單>個人檔案>產品授權>主畫面右上角】點擊【新增授權】
* 授權序號，填入●●●●●●●●●●●●●
* 註冊序號，填入●●●●●●●●●
* 即可取得license並在【產品支援網站>左側主選單>下載】下載相關安裝檔

3\. 取得合約

* 登入後於【左側主選單>個人檔案>維運合約>主畫面右上角】點擊【新增合約】
* 授權序號，填入●●●●●●●●●●●●●

> 授權序號及註冊序號需購買產品才能取得。
