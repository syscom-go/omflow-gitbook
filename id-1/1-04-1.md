## 最新版本

#### Release Candidate 1.2.6.0 (2026-03-01)

* **系　　統**：主選單新增【通知設定】功能
* **系　　統**：修正安裝檔於使用 MySQL 資料庫時可能導致安裝失敗之問題
* **服務請求**：修正日期欄位設定為唯讀後仍可被修改之問題
* **服務請求**：於手機支援設定中，新增【僅支援已綁定手機】選項
* **資料收集**：隱藏資料收集器流程設計中非必要之元件
* **資料收集**：修正文字類型資料介面顯示異常之問題
* **報　　表**：修正流程與其子流程名稱相同時，無法產生報表之問題
* **應用管理**：篩選元件新增【使用者編號轉換為使用者名稱、部門編號轉換為部門代號】功能
* **應用管理**：程式碼元件（Tools）新增【職務權責】相關功能
* **應用管理**：開始、人工、結束元件新增【通知】頁籤，以支援個人通知發送
* **應用管理**：更新表單設計之欄位新增方式
* **應用管理**：服務水準（SLA）觸發燈號時，調整為發送個人通知
* **簽　　核**：修正查找簽核人之 Python 程式碼未符合規範仍繼續執行之問題
* **一般表單**：靜態表單之【匯入】機制將依據 `data_no` 欄位判斷資料為新增或更新
* **一般表單**：修正流程檢視中，遮罩欄位經【資料驗證】處理後失效之問題
* **一般表單**：修正關聯單檢視時，【檔案上傳欄位】顯示異常之問題
* **人員管理**：新增【通知設定】頁籤，可設定個人通知接收方式
* **多國語言**：修正部分日文介面文字顯示
* **系統設定**：新增【DevTool 除錯】選項，預設為關閉
* **系統設定**：當 LDAP 自動同步失敗時，可設定通知指定人員

> 注意事項：  
> 1. 自此版本起取消試用授權（Free License）  
> 2. 因應資安政策更新，本產品自此版本起須使用 Python `3.11`
